================================================================================
4. REQUIREMENT: "Try to display the item in the frontend (via shortcode)"
================================================================================
STATUS: COMPLETED ✅

IMPLEMENTATION DETAILS:
- Engineered a "Direct Replacement Engine" inside 'hook_node_view'.
- Why this approach? 
  - It bypasses restricted Drupal Text Filters (which often strip brackets).
  - It is 100% stable and prevents server recursion crashes.
- Syntax Support:
  - [book_cards] : Lists all library items in a premium grid.
  - [book_cards id="X"] : Displays a specific book card.
  - [book_cards id="X,Y,Z"] : Displays a curated selection of books.
- Design: Applied a high-end Glassmorphism system (css/book-cards.css) 
  feat. Backdrop Blur, Luminous Glows, and Capsule Shimmer Buttons.
